Buncn is a lightweight monorepo template built on top of Bun & Shadcn/ui.

It comes with a seamlessly integrated stack:

- [Bun](https://bun.sh)
- [Shadcn/ui](https://ui.shadcn.com)
- [React Router](https://reactrouter.com)
- [Drizzle ORM](https://drizzleorm.com)
- [TanStack Query](https://tanstack.com/query/v5)
- [Better Auth](https://betterauth.dev)
- [Biome](https://biomejs.dev)

It aims to provide a solid foundation for building web applications with the most common features out of the box.
